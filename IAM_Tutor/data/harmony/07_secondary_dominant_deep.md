# [Deep Dive] 화성학: 세컨더리 도미넌트(Secondary Dominant)의 실전 활용

## 1. 세컨더리 도미넌트의 개념
다이아토닉(Diatonic) 코드 진행에서, 목적지 코드를 일시적인 '토닉(으뜸화음)'으로 간주하고, 그 목적지로 향하는 5도 위의 도미넌트 세븐스 코드를 강제로 삽입하는 기법입니다.

### 1.1 해결의 원리 (V7 -> I)
도미넌트 세븐스 코드가 가진 트라이톤(Tritone)의 극심한 불안정성이 완전협화음으로 해결되려는 성질을 이용해 진행에 강력한 추진력을 더합니다.

## 2. 곡에서의 실제 적용 사례

### 2.1 C Major 곡에서의 예시
기본 진행: C - Am - Dm - G7

*   **V7/VI 적용**: Am로 향하는 도미넌트 E7을 삽입합니다.
    *   변경: C - **E7** - Am - Dm - G7
    *   효과: E7의 G# 음표가 논-다이아토닉 음표로서 색채감을 급격히 변화시키며, Am로 떨어질 때의 비극적인 느낌을 강화합니다.
*   **V7/II 적용**: Dm로 향하는 도미넌트 A7을 삽입합니다.
    *   변경: C - E7 - Am - **A7** - Dm - G7

## 3. 릴레이티드 투 마이너 (Related II-m7)
세컨더리 도미넌트 앞에 그 도미넌트를 V7으로 하는 II-m7 코드를 짝지어 분할하는 고급 편곡 기법입니다.
*   예 (A7 - Dm 진행): **Em7(b5)** - A7 - Dm 로 확장 가능.
