# IAM Tutor (IndustrialAI-Music Tutor) 개발 계획서

## 1. 프로젝트 개요
기타 학습자를 위한 **사용자 입력 기반 지능형 화성학 분석 및 수준별 연주 가이드** AI 에이전트입니다.
정확도가 낮은 자동 추출 대신, 사용자가 직접 입력한 코드 진행이나 악보 이미지를 바탕으로 RAG 지식 베이스를 활용하여 심층적인 음악적 원리(Why)를 설명하고 수준별 맞춤 가이드를 제공합니다.

## 2. 폴더 구조
```
IAM_Tutor/
├── app.py                  # Streamlit 메인 UI (코드 입력 및 이미지 업로드)
├── agent/
│   ├── core.py             # ReAct 에이전트 정의 (화성학 튜터 페르소나)
│   └── tools.py            # 화성학 분석 도구 (기능 분석, 기법 탐지, 연주 가이드)
├── rag/
│   ├── vectorstore.py      # FAISS 인덱스 관리
│   ├── loader.py           # YAML 메타데이터 파싱 지원 로더
│   └── config.py           # RAG 관련 설정
├── data/                   # RAG용 지식 베이스 (YAML 메타데이터 포함)
│   ├── technique/          # 연주 기술 문서
│   └── harmony/            # 화성학 이론 문서
├── [Design_Docs]/          # 설계 문서
├── .env                    # API Key 관리
└── requirements.txt
```

## 3. 기술 스택
| 영역 | 기술 | 용도 |
|------|------|------|
| **LLM** | ChatGoogleGenerativeAI (Gemini 3.5 Flash) | 메인 추론 및 멀티모달 분석(Vision) |
| **Agent** | LangChain ReAct Agent | 분석 도구 선택 및 논리적 추론 |
| **RAG** | FAISS + YAML Metadata | 수준별/주제별 정밀 검색 필터링 |
| **UI** | Streamlit | 텍스트/이미지 입력 기반 인터랙티브 인터페이스 |

## 4. 단계별 구현 계획

### Step 1: 지식 베이스(RAG) 고도화 (완료)
**목표**: 수준별 필터링이 가능한 정교한 화성학 코퍼스 구축
- **구조화**: `harmony/`, `technique/` 디렉터리 분리.
- **메타데이터**: 모든 문서에 YAML 프론트매터(level, topic, prereq) 적용.
- **입문 콘텐츠**: 초보자를 위한 기초 지식 문서 5종 보강.

### Step 2: ReAct 에이전트 및 화성학 분석 도구 개발 (완료)
**목표**: 정확한 데이터를 바탕으로 심층적인 분석 제공
- **도구 구현**:
  - `analyze_chord_progression`: 기능적 화성학 분석.
  - `detect_advanced_techniques`: 세컨더리 도미넌트, 모달 인터체인지 등 탐지.
  - `suggest_playing_guide`: 수준별 운지 및 연주 팁 제공.
- **추론 최적화**: RAG 데이터를 근거로 논리적 답변을 생성하도록 프롬프트 고도화.

### Step 3: 사용자 주도형 UI/UX 구현 (완료)
**목표**: 악보 입력 및 시각적 피드백 강화
- **입력 채널**: 텍스트 코드 진행 입력 및 악보 이미지 업로드(Vision 분석) 섹션.
- **시각화**: 4마디 단위 줄바꿈 코드 악보 출력 및 사고 과정 실시간 노출.
- **개인화**: 5단계 수준 진단 결과와 RAG 메타데이터의 유기적 결합.

### Step 4: 성능 안정화 및 시나리오 검증 (진행 중)
**목표**: 다양한 코드 진행에 대한 분석 정확도 및 안정성 확보
- **예외 처리**: 분석 실패 시 우회 답변 방지 및 503 에러 재시도 로직 강화.
- **검증**: 입문/중급/상급 시나리오별 테스트 케이스 수행.
