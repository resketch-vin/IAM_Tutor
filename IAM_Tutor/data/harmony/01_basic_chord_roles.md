# [Deep Dive] 화성학: 기능적 화성법과 대리 화음의 미시적 분석

## 1. 코드 기능론의 심화 (Microscopic Functional Analysis)

음악의 흐름을 결정하는 세 가지 핵심 기능(Tonic, Sub-Dominant, Dominant)을 단순한 분류를 넘어 화성적 에너지의 관점에서 분석합니다.

### 1.1 토닉 (Tonic, I) - 정적 에너지의 중심
*   **물리적 기초**: 으뜸음(Root)과 완전5도가 형성하는 가장 견고한 배음 구조.
*   **확장 기능**: 
    *   **IMaj7**: 7도음(Leading Tone)이 포함되어 안정적이지만 선율적인 긴장감을 내포함.
    *   **VIm7 (대리 토닉)**: 으뜸음과 3도, 5도를 공유(C Major 기준 C, E, G vs A, C, E). 단조의 색채를 빌려오며 하행 진행의 핵심 역할을 함.
    *   **IIIm7 (대리 토닉)**: 3도, 5도, 7도를 공유. 토닉의 성격과 도미넌트로 넘어가는 징검다리 성격(C Major 기준 E, G, B)을 동시에 가짐.

### 1.2 서브도미넌트 (Sub-Dominant, IV) - 동적 팽창
*   **물리적 기초**: 으뜸음에서 4도 상행(또는 5도 하행)하여 조의 범위를 확장함.
*   **고급 활용**:
    *   **IVMaj7**: 스케일의 4도음을 근음으로 하여 도미넌트로 가기 위한 추진력을 생성.
    *   **IIm7 (대리 서브도미넌트)**: IVMaj7보다 더 강한 추진력을 가짐. II-V-I 진행의 시작점으로, 근음이 5도 하행하며 도미넌트로 향하는 '강진행'을 형성함.

### 1.3 도미넌트 (Dominant, V) - 긴장의 정점과 해소
*   **Tritone의 원리**: V7 코드 내의 3도와 7도 음(C Major 기준 B와 F)은 증4도(Tritone) 관계이며, 이는 가장 불안정한 간격임. 이 음들이 각각 반음 상/하행하여 토닉의 으뜸음과 3도음으로 해결되려는 성질이 화성 진행의 핵심 동력임.
*   **V7sus4**: 3도 대신 4도를 사용하여 긴장을 일시적으로 유보하고 모호함을 유도함.

## 2. 고급 화성 기법 (Professional Harmonic Techniques)

### 2.1 세컨더리 도미넌트 (Secondary Dominant, V7/X)
다이아토닉 코드(I~VI)를 임시적 토닉으로 설정하고, 그에 대한 도미넌트를 사용하는 기법.
*   **V7/II (A7 -> Dm7)**: 부드러운 단조 느낌의 연결.
*   **V7/VI (E7 -> Am7)**: 강렬한 마이너 종지감 형성.
*   **사용 가능 텐션**: 해결되는 코드의 성격에 따라 Mixolydian, Mixolydian b9 b13, Altered 스케일 등을 선택적으로 사용함.

### 2.2 릴레이티드 IIm7 (Related IIm7)
모든 도미넌트 7화음 앞에 그에 대응하는 2도 마이너 코드를 배치하여 화성적 밀도를 높임.
*   **원리**: 단일 도미넌트 진행(G7 -> C)을 II-V 진행(Dm7 -> G7 -> C)으로 세분화.
*   **응용**: 세컨더리 도미넌트 앞에도 적용 가능 (Em7b5 -> A7 -> Dm7).

### 2.3 모달 인터체인지 (Modal Interchange) 심화 분석
평행조(Parallel Key)에서 화음을 차용하여 감정의 진폭을 조절함.
*   **IVm6 / IVm7 (Fm6, Fm7)**: C Major에서 가장 흔히 쓰이는 차용 화음. '아련하고 슬픈' 종지감 형성.
*   **bVI (AbMajor7)**: 웅장하고 압도적인 느낌.
*   **bVII (BbMajor7)**: 세련된 록/팝 사운드, 일명 'Backdoor Dominant' 진행의 핵심.

### 2.4 증4도 대리 화음 (Tritone Substitution, subV7)
V7의 Tritone(3, 7도)을 공유하는 다른 도미넌트 코드로 대체.
*   **구조**: G7 (B, F) <-> Db7 (F, B). 
*   **효과**: 베이스 라인이 Db -> C로 반음 하행하며 매우 부드럽고 재즈적인 해결감을 제공함.
