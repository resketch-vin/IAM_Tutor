# [Quick Guide] 코드 체인지(Chord Change)가 안 될 때 보는 꿀팁

코드를 바꿀 때 소리가 끊기거나 손가락이 꼬이는 것은 모든 초보자의 고민입니다. 이를 해결하는 3단계 전략을 소개합니다.

## 1. 앵커 핑거(Anchor Finger) 찾기
두 코드 사이에 **위치가 변하지 않는 손가락**이 있는지 확인하세요.
*   예: C 코드에서 Am 코드로 갈 때, 검지와 중지는 떼지 않고 약지만 옮기면 됩니다.
*   이 손가락을 기준점(Anchor)으로 삼으면 훨씬 빠르고 안정적으로 바꿀 수 있습니다.

## 2. 가이드 핑거(Guide Finger) 활용
줄 위에서 미끄러지듯 이동하는 손가락입니다.
*   예: D 코드에서 E7 코드로 갈 때, 약지를 2번 줄에서 떼지 않고 3프렛에서 0프렛 방향으로 미끄러뜨리며 다른 손가락을 잡습니다.

## 3. '공중 부양' 연습 (Air Changes)
코드를 잡은 상태에서 손가락을 지판에서 1~2cm 떼어 공중에 띄웁니다. 그 모양 그대로 다음 코드 모양으로 공중에서 바꾼 뒤 한꺼번에 착지합니다.
*   손가락을 하나씩 순서대로 짚는 버릇을 고치는 최고의 방법입니다.

## 4. 스트로크 4번 박자에서 미리 떼기
마디의 마지막 8분음표(4박의 '&')에서는 코드를 미리 떼고 다음 코드로 이동할 준비를 하세요. 이때 개방현 소리가 살짝 나도 리듬만 정확하다면 귀에는 자연스럽게 들립니다.
